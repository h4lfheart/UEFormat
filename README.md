## Features
### Model Data (.uemodel)
- LOD Models (Level of Detail)
- Vertices
- Indices
- Normals
- Texture Coordinates
- Vertex Colors
- Materials
- Weights
- Bones
- Sockets
- Morph Targets
- Collision Geometry (Convex Only)
- Virtual Bones

### Animation Data (.ueanim)
- Location Keys
- Rotation Keys
- Scale Keys
- Float Keys (Currently Unused)
